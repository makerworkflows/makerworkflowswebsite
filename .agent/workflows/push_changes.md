description: Push changes to GitHub
---

1. Check git status to ensure clean working directory or stage changes
   ```bash
   git status
   ```

2. Add changes
   ```bash
   git add .
   ```

// turbo
3. Commit changes (if any)
   ```bash
   git commit -m "Update website content"
   ```

// turbo
4. Push to remote
   ```bash
   git push
   ```
